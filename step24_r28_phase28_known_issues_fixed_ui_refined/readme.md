# 山海異獸誌 v2 - Step 8 穩定整合版

## 這一版的定位
這不是再往下疊新功能的版本，而是把目前已經打通的核心流程整理成：

- 可測
- 可交棒
- 可部署
- 可回滾

的穩定整合版。

## 已整合完成的主線
- 老師登入
- 依卡序 / token 載入學生
- 改名
- 儲存 AI 夥伴模式
- 老師直接加分（預覽 → 確認 → 寫入）
- 批量加分
- 商城實體商品 → voucher
- 老師兌現 voucher
- AI 夥伴對話（白澤 / 智者貓咪 / 中性助教）
- 學生核心流程測試
  - 每日答題
  - Boss 結算
  - 限制提示（daily-limit / battle-limit）

## 本版新增
- 穩定整合首頁文案
- 整合狀態面板
- 部署 / 維修 / 回滾說明區
- 回歸測試清單文件
- 交棒清單文件

## 啟動前必改
請先修改 `js/config.js` 內的：
- `apiKey`
- `messagingSenderId`
- `appId`

## 穩定整合版建議測試順序
1. 老師登入
2. 載入學生
3. 改名
4. 老師加分
5. 批量加分
6. AI 夥伴切換與對話
7. 商城 voucher → 老師兌現
8. 每日答題
9. Boss 測試
10. 看限制提示是否正確

## 下一步不是功能擴張，而是交棒決策
接下來最合理的是：
- 比對舊版哪些功能已可交棒
- 哪些功能暫留舊版
- 決定何時切換正式入口
